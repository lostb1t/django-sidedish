Templates

Template Tags
-------------

The following template tag libraries are available:

- `blocks`: tags for dealing with blocks in general

	blocks_area
	-----------

	Renders a block area

		{% blocks_area 'area_name' %}

	blocks_block
	-----------

	Renders a block

		{% blocks_block 'block_slug' %}
